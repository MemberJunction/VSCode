# Change Log

All notable changes to the "MemberJunction for Visual Studio Code" extension will be documented in this file.

## [0.1.0] - 2024-11-13

### Added
- **Entity Explorer**: Browse core and custom MemberJunction entities
- **IntelliSense**: Auto-completion for entity names in metadata JSON files
- **Hover Tooltips**: View entity details on hover
- **Real-time Validation**: Validate metadata files as you type
- **JSON Schema Support**: Schema-based validation and suggestions
- **Status Bar Integration**: Quick access to entity information
- **Commands**:
  - Refresh Entity Explorer
  - Validate Metadata File
  - Show Entity Information
  - Open Entity Definition

### Features
- Discovery of entities from `@memberjunction/core-entities`
- Discovery of custom entities from `packages/GeneratedEntities`
- Automatic workspace detection
- Configurable settings for all features
- Output channel for debugging and logging
